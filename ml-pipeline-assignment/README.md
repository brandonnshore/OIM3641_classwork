# Wine Quality Classification — ML Pipeline Comparison

  OIM 3641 Major Assignment: comparing PyCaret (low-code) and
  scikit-learn (manual) ML workflows, then deploying the best model
  as a FastAPI web service.

  ## Dataset

  UCI Wine Quality (white wine) — 4,898 samples, 11 chemical
  features, target = `quality` (integer 3–9).

  The original `quality` column is binarized into a binary target:
  - `1` ("good") = quality >= 7  (1,060 samples, ~22%)
  - `0` ("not good") = quality < 7  (3,838 samples, ~78%)

  ## Files

  - `discovery.py` — full comparison: PyCaret setup → compare_models →
    confusion matrix, then a manual scikit-learn re-implementation of
    the winning model with classification_report.
  - `main.py` — FastAPI app exposing a `/predict` endpoint that loads
    the saved PyCaret pipeline and returns predictions.
  - `best_pipeline.pkl` — the trained ExtraTrees model + preprocessing,
    saved by PyCaret.
  - `winequality-white.csv` — UCI dataset.
  - `Confusion Matrix.png` — generated by PyCaret for the best model.
  - `report.docx` — required screenshots and notes.

  ## Results

  PyCaret ranked the top 3 models as:
  1. Extra Trees Classifier — Accuracy 0.866, AUC 0.911, F1 0.62
  2. Random Forest Classifier — Accuracy 0.865
  3. LightGBM — Accuracy 0.863

  The manual scikit-learn rebuild of ExtraTrees produced 0.89 accuracy
  and 0.70 F1 on a single 30% holdout. The gap is methodological:
  PyCaret reports averages across 10 stratified CV folds (more
  conservative), while the manual run used a single split.

  ## Running it

  Create the conda environment and install dependencies:
  conda create -n wineml python=3.11 -y
  conda activate wineml
  pip install pycaret scikit-learn fastapi uvicorn pandas

  Run the comparison and train the model:
  python discovery.py

  Start the API:
  uvicorn main:app --reload

  Then visit `http://127.0.0.1:8000/docs` for the Swagger UI.

  ## Sample API request

  POST to `http://127.0.0.1:8000/predict` with this JSON body:

  ```json
  {
    "fixed_acidity": 7.0,
    "volatile_acidity": 0.27,
    "citric_acid": 0.36,
    "residual_sugar": 20.7,
    "chlorides": 0.045,
    "free_sulfur_dioxide": 45,
    "total_sulfur_dioxide": 170,
    "density": 1.001,
    "pH": 3.0,
    "sulphates": 0.45,
    "alcohol": 8.8
  }

  Response:

  {
    "prediction": 0,
    "label": "not good",
    "confidence": 1.0
  }

  See report.docx for a screenshot of the live Swagger UI returning
  this prediction.

  Save (`Cmd + S`).
